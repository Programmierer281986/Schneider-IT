Protokolle

IP

Hier lernen Sie alles über das Internet Protocol. Dieses Protokoll stellt die Grundlage des Internets da.

TCP

Hier lernen Sie alles über das Transmission Control Protocol. Es kümmert sich um eine verbindungsorientierte Datenübertragung zwischen Computer und Servern.

UDP

Hier lernen Sie alles über das User Datagram Protocol. Es kümmert sich um eine verbindungslose Datenübertragung zwischen Computer und Servern.

NAT

Hier lernen Sie alles über das Network Address Translation. Dieses Protokoll ist wesentlich, um ein lokales Netzwerk mit dem Internet zu verbinden.

ARP

Hier lernen Sie alles über das Address Resolution Protocol. Dieses Protokoll wandelt IP-Adressen in MAC-Adressen um und vice versa.

HTTP

Hier lernen Sie alles über das HyperText Transfer Protocol. Dieses Protokoll regelt die Datenübertragung zwischen Anwendungen, meistens im Internet.

HTTPS

Hier lernen Sie alles über das HyperText Transfer Protocol Secure. Dieses Protokoll ist wie HTTP, nur verschlüsselt.

DHCP

Hier lernen Sie alles über das Dynamic Host Configuration Protocol. Dieses Protokoll vergibt Rechnern in einem lokalem Netzwerk IP-Adressen.

DNS

Hier lernen Sie alles über das Domain Name System. Dieses Protokoll managed die Namensauflösung, weil Menschen sich Namen leichter merken.

FTP

Hier lernen Sie alles über das File Transfer Protocol. Dieses Protokoll bietet eine leichte Möglichkeit Dateien zu teilen.

TFTP

Hier lernen Sie alles über das Trivial File Transfer Protocol. Dieses Protokoll ist eine abgespeckte Version von FTP.

RDP

Hier lernen Sie alles über das Remote Desktop Protocol. Dieses Protokoll sorgt für eine funktionierende Remote-Verbindung.

SMTP

Hier lernen Sie alles über das Simple Mail Transfer Protocol. Dieses Protokoll managed den Ausgangsserver eines Mailservers.

POP3

Hier lernen Sie alles über das Post Office Protocol. Dieses Protokoll ist ein Eingangsserver (die E-Mails werden am Client gespeichert).

IMAP

Hier lernen Sie alles über das Internet Message Access Protocol. Dieses Protokoll ist ein moderner Eingangsserver (die E-Mails werden immer am Server gespeichert).

SSH

Hier lernen Sie alles über das Secure Shell. Dieses Protokoll wird verwendet, um lokal eine entfernte Kommandozeile verfügbar zu machen.

NTP

Hier lernen Sie alles über das Network Time Protocol. Dieses Protokoll sorgt für eine perfekte Synchronisierung von Echtzeituhren in Computersystemen.

ICMP

Hier lernen Sie alles über das Internet Control Message Protocol. Dieses Protokoll dient zum Austausch von Nachrichtenmeldungen und Fehlermeldungen.

NDP

Hier lernen Sie alles über das Neighbor Discovery Protocol. Dieses Protokoll tauscht Informationen zwischen einzelnen Host aus.
